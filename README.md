# Thesis - Compositional tool
This tool can be used to generate musical lines

## Concept

### 1. Mirror

Mirroring notes (-) on a not sounding center (=).

```
---
   ----        ----
       --------
===================
       --------
   ----        ----
---
```

### 2. Middle

Find the note (.) in the middle of the center (=) and mirror tone (-).

```
-------        ----
       --------
.......        ....
       ........ 
===================
       ........
.......        ....
       --------
-------        ----
```
